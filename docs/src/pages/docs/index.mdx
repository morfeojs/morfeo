Morfeo is a set of tools to build your **Design System** and **Style** your application, with nearly zero runtime.

## Understand Morfeo

To deeply understand what Morfeo is, and why you should use it in your workflow, you can think about Morfeo as a combination of 3 main parts:

<Steps style={{ position: 'relative' }}>

### ğŸ¨ Theming system

That makes you build your own theme (**Design Language**) or use the default one.

### ğŸ’… CSS-in-TS library

To _safely_ style your application based on your theme.

### ğŸ” Static Analysis

Morfeo will look at your code at bundle-time to collect all the `css-in-ts` you've written and then extract
only the CSS you need.

</Steps>

## Ready to start?

<Steps style={{ position: 'relative' }}>

### Write your component

```tsx showLineNumbers filename="Button.tsx" {1, 3-7} /classes('button')/ copy
import { morfeo } from '../morfeo';

const classes = morfeo.css({
  button: {
    bg: 'primary',
  },
});

function Button() {
  return <button className={classes('button')}>Hello, world</button>;
}
```

### Morfeo will figure out the right classes

```tsx showLineNumbers filename="Button.tsx" /className="bg-primary"/
function Button() {
  return <button className="bg-primary">Hello, world</button>;
}
```

### And generate only the needed CSS

```css showLineNumbers filename="morfeo.css" /.bg-primary/
.bg-primary {
  background-color: #0066ff;
}
```
